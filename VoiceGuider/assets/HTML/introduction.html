情景设定/日程表

进入情景模式，您可以根据不同的情景及个人需要，选择或者设定设和的模式
1. 情景设定是我们程序的主要模块，在其他非主页中按返回键可以快速回到情景模式的主页。
2. 进入情景设定，您可以添加自定义的情景模式，设定完成后按返回键即可保存您设定的模式。
在主页里的模式列表中，点击某个模式可以进行相应模式的主动切换。长按某个模式可以进行相应模式的编辑、删除或者设置默认操作（默认情景模式是在日程表无特殊设定时的情景模式，未设置时，保持用户原模式）。
3. 设置为默认的情景模式可以在您定时的情景模式结束后自动调用。
5. 进入到日程表任务设定界面，您可以建立任意个该天某个时间段的情景模式，保存之后在相应的开始时间我们会启用您设定的模式。
6.* 注意哦，时间不允许重叠（将提示无法创建）。

语音记事

进入语音记事，您可以在这里记下您的生活琐碎，点点滴滴或是灵感乍现。
用一句简单的文本，或者一个简短的录音，随时随地添加便签。
1. 在编辑框中输入相应文字后按【记事】按钮就可以将您记下的文字发送到界面上。
2. 按下【语音】按钮后，将开始录音，再按下【停止】按钮之后即可保存您刚才录下的内容。
3. 之后，可以通过按下语音标志的便笺图标，开始播放相应的录音内容。在播放的时候再次点击则可以停止录音播放。
4. 如果您想删除某条便笺，则长按相应的便笺就可以删除了。

定位情景

定位情景是我们智能情景模式的核心功能，您可以用地图定位将某个地方的地理位置（通过框定一个圆形区域）保存起来，然后设定这个位置下的相应的情景模式。当您到达该位置时软件将自动检测并触发改区域的事件，切换至相应的情景模式。
1. 进入定位情景，点击添加定位，进入地图定位界面。
2. 按下定位，您可以定位到您当前的位置。
3. 按下放大或者缩小按钮，可以改变您的定位的圆的半径大小。即是您要设定的位置的范围。
4. 点击离线地图，您可以选择进入相应省份设置您的定位。
5. 在地图上选择了您想要定位的位置之后,输入名称保存即可。
6. 返回主界面之后，您可以为您保存的地理位置设置相应的情景模式。
7. 在界面下勾选【启动定位】选项后，将开启本功能。

WIFI情景

我们的WIFI情景主要为了进一步完善情景模式“智能化”的这个主题，您可以将某个WIFI热点名称保存起来，然后设定这个WIFI下的相关的情景模式已经切换优先度。当您启用WIFI扫描的时候，我们的软件将自动为您检测周围环境的WIFI，如果检测到您保存的WIFI，我们将切换至这些WIFI中优先级别较高的热点相应的情景模式。
1. 打开WIFI后，点击WIFI列表。我们的软件将为您搜寻周围环境中的WIFI
2. 点击某个WIFI，我们就保存这个WIFI到WIFI列表中
3. 点击WIFI列表中的某个WIFI，则可以选择设定某个情景模式以及切换的优先级。
4. 在界面下勾选【启用WIFI切换】选项后，将开启本功能。
